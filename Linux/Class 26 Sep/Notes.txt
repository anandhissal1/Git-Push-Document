history -c : Clear/delete all commands registered in history

cat -n one : print file content with line numbers

head -n 15 : gives top 15 lines of a file

tail -n 12 : give me last 12 lines from file
tail -n +12 : give me 12th line onwards to last line from file

--------------------------------------------------------
Q) How to read log files data using tail command?
A) tail -f filename.log
   -f : display flow of new lines into log file
   (ctrl+C to come out of the file)


Q) how to know more about a command? Or open command manual?
A) use man command / press 'q' to come out of the manual

ex: man wc   (use arrows) , press q to exit


Q) how to print number of lines only / words only / characters only in a file?
A)
	wc -l filename  //lines count
	wc -w filename  // words count
	wc -c filename  // chars count

*) cp is used to create a copy of given file
Syntax: cp <source>  <destination>
ex:   cp  hello   mydetails/new_hello
      cat hello > mydetails/new_hello


Q) What is the command to rename or move a file?
A) mv

Syntax:  mv <source>  <destination>
ex:      mv  hello      hi   //rename file
         mv  one two three  mydetails/    //move all 3 files into mydetails


Q) How to install tree s/w in Linux ?
A) sudo yum install tree -y

tree <directory-name>
tree .
-------------------------------------------------
grep : used to search given word in given files

-i : ignore case
-n : print matching line numbers
*  : all files

examples:
grep 'data' welcome.txt : search word data in welcome.txt file
grep -i 'data' welcome.txt : search word data, DATA, Data, DatA ..all combinations
				in welcome.txt file
grep -n 'data' welcome.txt : search word data in welcome.txt file, display line numbers also
