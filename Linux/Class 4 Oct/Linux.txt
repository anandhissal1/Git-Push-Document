Website : Collection of WebPages(ex: LoginPage, RegisterPage, HomePage,
  ProfilePage, Settings Page, ..etc)
Ex: Facebook, Gmail, ..etc

*) Webpages are created using HTML,CSS and JavaScript.
 [Also some programming languages Java, .net, PHP ..etc]

*) httpd is a webserver software.
*) It is used to run Static Webpages (HTML,CSS,JS).
*) httpd works on port number : 80
*) We have to start (stop) server.
*) Dynamic WebPage means : Content of Webpage is different based on
		Users/locations/time.

[ Windows: services.msc -> Right click on service -> Start ]

----------------------------------------------------------------
Step#1 Install httpd server
sudo yum install httpd 

Step#2 We need to start httpd service

sudo systemctl start httpd
sudo systemctl stop httpd

sudo systemctl enable httpd [auto-start, when we restart/reboot linux server]
sudo systemctl status httpd  [press q to exit]
------------or--------------
sudo service httpd start
sudo service httpd status
sudo service httpd stop

Step#3 Test URL
Copy PublicIP from AWS console and enter URL like
  http://PublicIP:80  
  ex: http://65.2.187.188:80

Note: if it is not responding, open port 80(HTTP) in Security Group.

Step#4 Goto Webserver location and create one HTML file
> cd /var/www/html
> sudo vi index.html
 Press i to come into insert mode
