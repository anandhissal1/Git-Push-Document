Variable : memory block, name given to it, holds value.
key = val format.

Terraform supports 3 types of variables.
1. Input Variables
2. Local Variables
3. Output Variables

Project files:
1. main.tf : Actual configuration
2. variables.tf : declare variables, datatypes, default values 
3. terraform.tfvars : Variable with values
